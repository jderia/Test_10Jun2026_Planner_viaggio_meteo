
## Requisiti

Per usare il progetto servono:
- un computer
- una connessione;
- un browser, per esempio Chrome o Safari;
- Visual Studio Code per modificare, nel caso, i file;
- Python 3 per avviare il server locale.

# Installazione

Questo è un progetto fatto con HTML, CSS, JavaScript e Bootstrap.

## Scaricare il progetto

Per scaricare il progetto da GitHub:

git clone https://github.com/jderia/Test_10Jun2026_Planner_viaggio_meteo.git

## Aprire la cartella

Dopo il download bisogna entrare nella cartella:

cd Test_10Jun2026_Planner_viaggio_meteo

## Avviare il sito

Per far funzionare bene il sito puoi usare un server locale:

python3 -m http.server 8000

Poi apr il browser e scrivi:

http://localhost:8000

La pagina principale è index.html.

La pagina dove si vede il meteo è destinazioni.html.
