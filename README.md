# Planner viaggio con il meteo

Questo progetto è un piccolo sito per vedere il meteo di alcune città.

Le città scelte sono:

- Lisbona
- Atene
- Nairobi

## Tecnologie usate

Nel progetto ho usato:

- HTML
- CSS
- JavaScript
- Bootstrap
- API Open-Meteo

## Pagine del sito

Il sito ha tre pagine:

- index.html
- destinazioni.html
- consigli.html

La pagina destinazioni.html mostra i dati del meteo.

## Dati meteo

I dati vengono presi dalla API Open-Meteo.

Se la API non funziona, il sito usa i dati presenti nel file data.json.
